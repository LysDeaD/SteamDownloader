📚 Guide Complet SteamCMD

Tout ce que vous devez savoir sur SteamCMD et le téléchargement de mods

🤔 Qu'est-ce que SteamCMD ?

SteamCMD (Steam Console Client) est un outil en ligne de commande développé par Valve Corporation qui permet d'installer et de mettre à jour des applications Steam sans utiliser l'interface graphique traditionnelle de Steam. Cette solution est particulièrement utile pour les administrateurs de serveurs, les développeurs, et les utilisateurs avancés qui souhaitent automatiser le téléchargement de contenu Steam.

🔑 Fonctionnalités Clés

  • Téléchargement automatisé de jeux et mods
  • Installation de serveurs dédiés
  • Mise à jour automatique des applications
  • Gestion par scripts et automatisation
  • Fonctionnement sans interface graphique

💡 Pourquoi SteamCMD est-il Utile ?

SteamCMD présente plusieurs avantages significatifs par rapport au client Steam traditionnel. Tout d'abord, il consomme beaucoup moins de ressources système car il ne nécessite pas d'interface graphique. Cela le rend idéal pour les serveurs ou les environnements où les ressources sont limitées.

L'automatisation est l'un des principaux avantages de SteamCMD. Vous pouvez créer des scripts pour télécharger automatiquement des mods, mettre à jour des serveurs de jeux, ou gérer des collections entières de contenu Steam. Cette fonctionnalité est particulièrement précieuse pour les administrateurs de serveurs de jeux qui doivent maintenir leurs serveurs à jour avec les dernières versions des mods.

📥 Installation de SteamCMD

🪟 Windows

Étape 1 : Téléchargement

Téléchargez SteamCMD depuis le site officiel de Valve :

https://developer.valvesoftware.com/wiki/SteamCMD

Étape 2 : Extraction

Créez un dossier (ex: C:\steamcmd) et extrayez le fichier steamcmd.zip dans ce dossier.

Étape 3 : Premier lancement

Ouvrez une invite de commande, naviguez vers le dossier SteamCMD et exécutez :

steamcmd.exe

SteamCMD se mettra à jour automatiquement lors du premier lancement.

🐧 Linux

Ubuntu/Debian

sudo apt update sudo apt install steamcmd

Installation manuelle

mkdir ~/steamcmd cd ~/steamcmd wget https://steamcdn-a.akamaihd.net/client/installer/steamcmd_linux.tar.gz tar -xvzf steamcmd_linux.tar.gz ./steamcmd.sh

⚙️ Comment Fonctionne SteamCMD ?

SteamCMD fonctionne en se connectant directement aux serveurs de contenu de Steam via une interface en ligne de commande. Il utilise le même système d'authentification que Steam, ce qui signifie que vous pouvez vous connecter avec votre compte Steam ou utiliser une connexion anonyme pour certains types de contenu.

🔄 Flux de Travail Typique

  1. Lancement de SteamCMD
  2. Connexion (anonyme ou avec compte)
  3. Sélection du répertoire d'installation
  4. Téléchargement du contenu souhaité
  5. Fermeture de SteamCMD

📖 Commandes de Base

Connexion

Connexion anonyme

login anonymous

Utilisé pour télécharger du contenu public comme les serveurs dédiés.

Connexion avec compte

login nom_utilisateur mot_de_passe

Nécessaire pour télécharger des mods ou du contenu privé.

Téléchargement

Définir le répertoire d'installation

force_install_dir C:\MonJeu\

Télécharger une application

app_update 108600 validate

Télécharge Project Zomboid (ID: 108600) et vérifie l'intégrité des fichiers.

Télécharger un mod Workshop

workshop_download_item 108600 2619072426

Télécharge le mod ID 2619072426 pour Project Zomboid.

🎯 Exemples Pratiques

Télécharger un mod Project Zomboid

steamcmd.exe login anonymous force_install_dir C:\PZMods\ workshop_download_item 108600 2619072426 quit

Script automatisé (Windows .bat)

@echo off cd /d "C:\steamcmd" steamcmd.exe +login anonymous +force_install_dir "C:\PZMods\" +workshop_download_item 108600 2619072426 +quit pause

🔧 Techniques Avancées

📦 Téléchargement en Lot

Télécharger plusieurs mods

Créez un script pour télécharger plusieurs mods en une fois :

@echo off set STEAMCMD_PATH=C:\steamcmd set INSTALL_DIR=C:\PZMods set APP_ID=108600 cd /d "%STEAMCMD_PATH%" REM Liste des IDs de mods set MODS=2619072426 2696083206 2169435993 for %%i in (%MODS%) do ( echo Téléchargement du mod %%i... steamcmd.exe +login anonymous +force_install_dir "%INSTALL_DIR%" +workshop_download_item %APP_ID% %%i +quit ) echo Tous les mods ont été téléchargés ! pause

🔄 Automatisation avec Planificateur

Mise à jour automatique

Créez un script de mise à jour automatique :

@echo off cd /d "C:\steamcmd" steamcmd.exe +login anonymous +force_install_dir "C:\MonServeur\" +app_update 108600 validate +quit REM Redémarrer le serveur après mise à jour taskkill /F /IM "ProjectZomboid64.exe" start "Serveur PZ" "C:\MonServeur\ProjectZomboid64.exe"

🔍 Vérification et Validation

Vérifier l'intégrité des fichiers

app_update 108600 validate

Le paramètre validate vérifie tous les fichiers et re-télécharge ceux qui sont corrompus.

📊 Gestion des Logs

Activer les logs détaillés

steamcmd.exe +@ShutdownOnFailedCommand 1 +@NoPromptForPassword 1 +login anonymous +force_install_dir "C:\MonJeu\" +app_update 108600 validate +quit > update_log.txt 2>&1

🌍 Gestion Multi-Serveurs

Configuration pour plusieurs serveurs

REM Serveur 1 steamcmd.exe +login anonymous +force_install_dir "C:\Serveur1\" +app_update 108600 validate +quit REM Serveur 2 steamcmd.exe +login anonymous +force_install_dir "C:\Serveur2\" +app_update 108600 validate +quit REM Mise à jour des mods pour tous les serveurs for /d %%i in (C:\Serveur*) do ( steamcmd.exe +login anonymous +force_install_dir "%%i" +workshop_download_item 108600 2619072426 +quit )

🔧 Problèmes Fréquents et Solutions

❌ Erreur "Login Failure" (Échec de Connexion)

Solutions possibles :

  • Vérifiez vos identifiants Steam
  • Désactivez temporairement l'authentification à deux facteurs
  • Utilisez un code d'authentification Steam Guard
  • Essayez "login anonymous" pour le contenu public
login nom_utilisateur mot_de_passe CODE_STEAM_GUARD

❌ Mod Non Trouvé ou Inaccessible

Vérifications :

  • Vérifiez que l'ID du mod est correct
  • Assurez-vous que le mod est public
  • Certains mods nécessitent une connexion avec votre compte

❌ Erreur "Disk Write Error"

Solutions :

  • Vérifiez l'espace disque disponible
  • Lancez SteamCMD en tant qu'administrateur
  • Vérifiez les permissions du dossier de destination
  • Changez le répertoire d'installation

❌ Téléchargement Lent ou Bloqué

Optimisations :

  • Changez de région de téléchargement
  • Redémarrez SteamCMD
  • Vérifiez votre connexion internet
  • Utilisez un VPN si nécessaire
@sSteamCmdForcePlatformType windows @sSteamCmdForceInstallType server

❌ Problèmes de Mise à Jour

Solutions :

  • Utilisez toujours le paramètre validate
  • Supprimez le cache SteamCMD : steamcmd/steamapps/
  • Réinstallez SteamCMD si nécessaire
